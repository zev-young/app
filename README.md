# apps
这个仓库用于存放开源的安卓应用

## NoAds
用于自动点击跳过安卓应用开屏广告

使用方法如下：
1. 下载并安装NoAds.apk到安卓设备上
2. 进入"设置"->"无障碍"->"已下载的应用"->"屏蔽广告"开启
3. 防止被系统杀死，可在"设置"->"应用设置"->"NoAds"->"省电策略"->"无限制"
